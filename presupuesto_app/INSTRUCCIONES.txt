CÓMO USAR LA APLICACIÓN:
========================

OPCIÓN 1 - Abrir directamente en el navegador:
  - Abra el archivo index.html en Chrome/Firefox de su Android
  - La app funciona sin internet una vez cargada

OPCIÓN 2 - Instalar como APK/App en Android:
  1. Suba los archivos a un hosting web (o use GitHub Pages gratis)
  2. Abra la URL en Chrome para Android
  3. Chrome mostrará "Agregar a pantalla de inicio"
  4. Toque instalar → La app queda instalada como APK nativa

OPCIÓN 3 - Hosting gratuito con GitHub Pages:
  1. Vaya a github.com y cree una cuenta gratuita
  2. Cree un nuevo repositorio público
  3. Suba todos los archivos de esta carpeta
  4. En Settings > Pages, active GitHub Pages
  5. Su URL será: https://su-usuario.github.io/nombre-repo/
  6. Abra esa URL en Chrome Android e instale
